# 🖌️ Random Street Artist

This project's aim was to teach us modularity in Python, I made it back when I was 16. It draws 4 randomly generated buildings using the turtle Python module. Every aspect of the building is randomized.

### Randomized elements:

* Number of floors (from 1 to 5)
* Building color
* Door style (squarred/rounded)
* Door position
* Door color
* Windows (regular window/balcony)
* Roof style (triangular/flat)
